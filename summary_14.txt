Modal or non - modal Window
47
Modaliniai  ir nemodaliniai
When a window is  modal it remains active and 
focused until the user has finished with it and 
dismisses it.  While it is active no other windows of the 
same application can be activated. 
Modaliniai ,  jei Sąveika su kitais langais yra  uždrausta
So a non - modal window is the opposite. While it is 
active you can still activate other windows. The user 
can switch between windows of the same 
application. The window being active does not 
prevent the rest of the application from continuing.
Nemodaliniai,  jei Sąveika  su kitais langais yra galima
